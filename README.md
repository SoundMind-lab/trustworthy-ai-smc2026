# Trustworthy AI Workshop Website

This folder contains a simple professional multi-page website for a workshop at IEEE SMC 2026.

## Files
- `index.html` — home page
- `cfp.html` — call for papers
- `submission.html` — submission guidelines
- `program.html` — workshop program
- `organizers.html` — organizers page
- `contact.html` — contact page
- `styles.css` — shared styling

## How to publish with GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder to the repository root.
3. Open the repository on GitHub.
4. Go to **Settings > Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select branch **main** and folder **/(root)**.
7. Save.
8. After GitHub finishes publishing, your website URL will look like:
   `https://your-username.github.io/repository-name/`

## Before publishing
Update these placeholders:
- Organizer names and affiliations in `organizers.html`
- Contact email in `contact.html`
- Submission portal link in `submission.html`
- Any workshop date/location details you want to add
